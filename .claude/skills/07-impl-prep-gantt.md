# 實作準備階段 - 人力甘特圖

## 待補充問題（逐一確認後完善此 SKILL）

### 關於甘特圖格式
- 甘特圖用什麼工具產出？Google Sheets、xlsx、Mermaid、或其他？
- 粒度：週為單位還是天？
- 是否需要標示里程碑（M1/M2/M3）？

### 關於人力安排
- 人員是否為固定班底還是每次重新分配？
- 多個專案並行時，人力衝突如何在甘特圖中顯示？
- 兼任角色（如 DevOps/MIS 兼任）如何標記可用時間？

### 關於時程計算
- 假日/例假日是否自動排除？
- 預留的緩衝時間（buffer）通常設多少？
- 客戶審核時間（等待期）如何在甘特圖中呈現？

### 關於更新機制
- 甘特圖是靜態文件還是動態追蹤（隨實際進度更新）？
- 甘特圖變更時是否需要通知客戶？
- 與 GitLab milestone 是否需要同步？
