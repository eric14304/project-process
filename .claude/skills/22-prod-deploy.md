## 待補充問題（逐一確認後完善此 SKILL）

### 關於 PROD 部署
- PROD 部署時間窗口（通常選非尖峰時段，幾點？）？
- 部署前需要客戶 IT 配合什麼（DNS 切換 / 憑證更新 / 防火牆）？
- PROD 環境（國泰案：4台 VM）是否由客戶 IT 自行維護還是我方協助？
- 藍綠部署 / 滾動更新還是停機部署？

### 關於正式服務串接
- 需要確認串接的正式服務清單（TW DIW 正式環境 / 阜爾正式門禁 / 其他）？
- 串接確認的測試範圍（smoke test 清單）？
- 若串接失敗的 rollback 計畫？

### 關於上線驗收
- 上線驗收項目清單由誰定義（PM / 客戶 / 合約規定）？
- 驗收簽核方式（書面 / email）？
- 上線後的保固期（通常幾個月）與保固內容？

### 關於後續維護
- 維護費用與 SLA 是否在合約中定義？
- 上線後 on-call 機制（誰接、反應時間）？
- 定期備份、版本更新的負責方？
